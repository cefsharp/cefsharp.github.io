WindowInfoSetAsWindowless Method

Version 92.0.260
Create the browser using windowless (off-screen) rendering. No window will be created for the browser and all rendering will occur via the CefRenderHandler interface. This window will automatically be transparent unless a colored backgrond is set in the browser settings.

Parameters

parentHandle
Type: SystemIntPtr
Value will be used to identify monitor info and to act as the parent window for dialogs, context menus, etc. If not provided then the main screen monitor will be used and some functionality that requires a parent window may not function correctly. In order to create windowless browsers the CefSettings.windowless_rendering_enabled value must be set to true.
