LoadHandlerOnLoadError Method

Version 101.0.180
Called when the resource load for a navigation fails or is canceled. ErrorCode is the error code number, ErrorText is the error text and FailedUrl is the URL that failed to load. See net\base\net_error_list.h for complete descriptions of the error codes. This method will be called on the CEF UI thread. Blocking this thread will likely cause your UI to become unresponsive and/or hang.
