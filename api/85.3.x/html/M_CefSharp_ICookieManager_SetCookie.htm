ICookieManagerSetCookie Method

Version 85.3.121
Sets a cookie given a valid URL and explicit user-provided cookie attributes. This function expects each attribute to be well-formed. It will check for disallowed characters (e.g. the ';' character is disallowed within the cookie value attribute) and fail without setting the cookie if such characters are found. This method will be executed on the CEF UI thread in an async fashion, to be notified upon completion implement ISetCookieCallback and pass in as callback
