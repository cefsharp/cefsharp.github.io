GetMimeTypeDelegate
GetMimeType(String) is being deprecated in favour of using Chromiums native mimeType lookup which is accessible using Cef.GetMimeType, this method is however not directly available as it exists in CefSharp.Core, to get around this we set this static delegate with a reference to Cef.GetMimeType when Cef.Initialize is called.
