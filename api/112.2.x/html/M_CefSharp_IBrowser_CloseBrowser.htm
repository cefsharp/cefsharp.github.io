Parameters

forceClose
Type: SystemBoolean
If forceClose is false the event handler, if any, will be allowed to prompt the user and the user can optionally cancel the close. If forceClose is true the prompt will not be displayed and the close will proceed. Results in a call to DoClose(IWebBrowser, IBrowser) if the event handler allows the close or if forceClose is true See DoClose(IWebBrowser, IBrowser) documentation for additional usage information.
