OnBeforePopupCreated The LifeSpanHandlerOnBeforePopupCreatedDelegate will be called before the popup has been created and can be used to cancel popup creation if required, modify IBrowserSettings and disable javascript.
OnPopupBrowserCreated The LifeSpanHandlerOnPopupBrowserCreatedDelegate will be called when theIBrowser has been created. The IBrowser instance is valid until OnPopupDestroyed(LifeSpanHandlerOnPopupDestroyedDelegate) is called. IBrowser provides low level access to the CEF Browser, you can access frames, view source, perform navigation (via frame) etc.
OnPopupCreated The LifeSpanHandlerOnPopupCreatedDelegate will be called when theChromiumWebBrowser has been created. When the LifeSpanHandlerOnPopupCreatedDelegate is called you must add the control to it's intended parent.
OnPopupDestroyed The LifeSpanHandlerOnPopupDestroyedDelegate will be called when the ChromiumWebBrowser is to be removed from it's parent. When the LifeSpanHandlerOnPopupDestroyedDelegate is called you must remove/dispose of the ChromiumWebBrowser.
