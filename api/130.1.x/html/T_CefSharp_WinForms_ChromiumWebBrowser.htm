ChromiumWebBrowserImportant!!! This constructor exists as the WinForms designer requires a parameterless constructor, if you are instantiating an instance of this class in code then use the ChromiumWebBrowser(String, IRequestContext) constructor overload instead. Using this constructor in code is unsupported and you may experience NullReferenceException's when attempting to access some of the properties immediately after instantiation.
