AccessibilityHandler Provides a limited read-only Accessibility Handler implementation. To enable accessibility support use the --force-renderer-accessibility to enable for all browsers or call SetAccessibilityState(CefState) on a per browser basis to enable. By default accessibility is disabled by default. Having accessibility enabled can impact performance until accessibility is disabled.
