ChromiumWebBrowser(HwndSource, String, Size) Initializes a new instance of the ChromiumWebBrowser class. Use this constructor to load the browser before it's attached to the Visual Tree. The underlying CefBrowser will be created with the specified size. CEF requires positive values for Width and Height, if values less than 1 are specified then the default value of 1 will be used instead.
