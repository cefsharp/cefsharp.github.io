Interfaces
 InterfaceDescription
IAsyncMethodInterceptor Provides the capability intercept async/sync Net method calls made from javascript as part of the JavascriptBinding (JSB) implementation. One example use case is logging method calls. Extends IMethodInterceptor to add async support.
IBinder Converts input params into complex .Net types (can also be used for type conversion). This feature is similar in concept to ASP.NET MVC Model Binding. Objects passed from javascript are represented as IDictionaryTKey, TValue and arrays/lists as IListT See DefaultBinder for the default implementation.
IMethodInterceptor Provides the capability intercept Net method calls made from javascript as part of the JavascriptBinding (JSB) implementation. One example use case is logging method calls.
IPropertyInterceptor Provides the capability intercepting get/set property calls made from javascript as part of the JavascriptBinding (JSB) implementation.
