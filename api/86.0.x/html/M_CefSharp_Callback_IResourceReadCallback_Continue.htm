IResourceReadCallbackContinue Method

Version 86.0.240
Callback for asynchronous continuation of Read(Stream, Int32, IResourceReadCallback). If bytesRead == 0 the response will be considered complete.

Parameters

bytesRead
Type: SystemInt32
If bytesRead == 0 the response will be considered complete. If bytesRead > 0 then Read(Stream, Int32, IResourceReadCallback) will be called again until the request is complete (based on either the result or the expected content length). If bytesRead < 0 then the request will fail and the bytesRead value will be treated as the error code.
